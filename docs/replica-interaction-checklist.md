# 相弈象棋复刻：交互验收清单

版本：v0.1  
日期：2026-04-28

## 1. 全站导航

| 交互 | 桌面端要求 | 移动端要求 | 验收 |
| --- | --- | --- | --- |
| 帮助下拉 | 点击/hover 展开，显示源站同顺序帮助项 | 可点击展开，不遮挡正文 | 每项链接到本地复刻页 |
| 语言下拉 | 展开简中、繁中、English、Tiếng Việt | 可点击展开，当前语言清晰 | 跳转本地语言页，不跳源站 |
| 下载 APP | 跳转本地下载页 | 同桌面 | 链接与焦点状态正常 |
| 如何下象棋 | 跳转本地教程页 | 同桌面 | 保留 SEO canonical |
| 在线对弈 CTA | 当前页跳转 `https://play.xiangqi.com/` | 同桌面 | 允许外链，不新开窗口 |

## 2. 首页

| 模块 | 交互要求 | 验收 |
| --- | --- | --- |
| 首屏 CTA | 在线对弈外跳，下载 App 进入本地下载页 | 点击路径正确 |
| 教学文章 | 卡片 hover、点击进入本地文章详情、更多进入本地列表 | 不出现死链 |
| FAQ | 点击问题展开答案，再次点击收起，箭头/图标状态变化 | 键盘可聚焦更佳 |
| 内链 | SEO 正文中的“这里/下载/规则”等链接按规则指向本地或允许外链 | 不跳回源站普通页面 |

## 3. 帮助中心

| 模块 | 交互要求 | 验收 |
| --- | --- | --- |
| 帮助搜索 | 若源站有搜索框，需实现本地过滤或同等展示效果 | 输入关键词后列表变化 |
| 帮助分类 | 点击进入本地帮助详情页 | 每个帮助项可访问 |
| 详情页目录 | 若源页有目录/锚点，点击滚动到对应段落 | URL hash 或滚动位置正确 |
| 图片教程 | 图文步骤顺序和源站一致 | 图片正常显示 |

## 4. 教程与文章

| 模块 | 交互要求 | 验收 |
| --- | --- | --- |
| 目录锚点 | 点击目录滚动到对应标题 | 桌面/移动端都可用 |
| 文章卡片 | hover 和点击状态复刻源站 | 点击进入详情页 |
| 文章详情 | 返回列表、上下篇、内部链接可用 | 链接路径正确 |
| 多语言切换 | 切到对应语言同主题页面；若无同主题页，切到对应语言首页并记录 | 不直接跳外部语言站 |

## 5. FAQ

| 模块 | 交互要求 | 验收 |
| --- | --- | --- |
| 首页 FAQ | 展开/收起真实答案 | 不只静态文本 |
| FAQ 列表页 | 支持多个问题展开；如源站只允许单开，需跟源站一致 | 状态与源站一致 |
| FAQ 内链 | 本地页面优先，允许业务外链 | 无乱码、无死链 |
| 移动端 | 展开后高度自适应，不遮挡后续内容 | 390px 截图检查 |

## 6. 下载页

| 模块 | 交互要求 | 验收 |
| --- | --- | --- |
| APK 下载 | 保留真实下载链接，按源站效果跳转 | 点击地址正确 |
| Google Play | 保留真实外链和源站图标，按源站效果跳转 | 图标一致 |
| App Store | 保留真实外链和源站图标，按源站效果跳转 | 图标一致 |
| 下载 FAQ | 如源站可展开，需实现展开/收起 | 移动端不溢出 |

## 7. 多语言

| 语言 | 验收要求 |
| --- | --- |
| 简中 | 作为像素级首页主基准 |
| 繁中 | 页面结构、图片、交互和移动端按繁中源站复刻 |
| English | 页面结构、帮助菜单、文章/FAQ、资源按英语源站复刻 |
| Tiếng Việt | 页面结构、SEO长文、文章/FAQ、下载页按越南语源站复刻 |

## 8. 错误状态

- 404 页面需本地化，不跳源站。
- 外部链接失败不影响静态站构建。
- 图片缺失时构建阶段应能发现；验收不得通过空白首屏。
