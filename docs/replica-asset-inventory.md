# 相弈象棋复刻：资源清单与本地化规则

版本：v0.1  
日期：2026-04-28

## 1. 资源本地化原则

- 页面渲染所需的图片、背景图、图标、字体、CSS、JS 必须放入本地项目，不依赖源站运行时。
- 允许保留的外链仅限业务动作：`https://play.xiangqi.com/`、真实 APK / Google Play / App Store 下载链接、源站实际社交媒体链接。
- 繁中、英语、Tiếng Việt 页面执行与简中同等级别的资源本地化要求。
- 资源抓取必须覆盖 `<img src>`、`original`、`data-src`、`srcset`、CSS `background-image`、内联 JSON、favicon、商店徽章、社交图标。
- 不使用转码损坏的 HTML 作为页面源码；抓取时保留原始字节并按响应头或页面 `charset` 解码。

## 2. 已有本地资源目录

| 目录 | 当前用途 | 下一步 |
| --- | --- | --- |
| `public/mirror/` | 已同步一批源站图片资源 | 建立 URL 到本地文件映射，删除未使用项前先确认 |

## 5. 资源映射表字段

后续建立 `docs/replica-asset-map.csv`，字段如下：

```csv
id,locale,page_id,source_url,source_attr,asset_type,local_path,width,height,status,notes
```

状态枚举：

- `found`：已在源页面发现。
- `downloaded`：已保存到 `public`。
- `mapped`：页面代码已引用本地路径。
- `verified`：浏览器截图确认显示正常。
- `replace-needed`：当前本地资源不一致，需要替换。

## 6. 禁止项

- 禁止用空白占位图替代源站图片。
- 禁止用自制 SVG 替代源站 logo、商店徽章、社交图标，除非源站本身就是 SVG 且已本地化。
- 禁止依赖浏览器从源站拉取 `original` 懒加载图片。
- 禁止图片显示正常但文字乱码；资源验收必须和编码验收一起完成。
